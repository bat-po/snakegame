# 🐍 Snake Game

A classic Snake game built with HTML5 Canvas, CSS3, and Vanilla JavaScript.

## 🎮 How to Play

1. Open `index.html` in your web browser
2. Use **Arrow Keys** or **WASD** to control the snake
3. Eat the red food to grow and score points
4. Avoid hitting walls or yourself
5. The game speeds up as you score higher!

## 🎯 Controls

- **Arrow Keys** or **WASD**: Move snake
- **Space/Enter**: Restart after game over

## 🏆 Scoring

- Each food item: **+10 points**
- Game speed increases every 50 points
- High score is saved in browser storage

## 🎨 Features

- Smooth animations and modern design
- Snake with eyes that follow direction
- Food with glowing effect
- Score tracking with high score persistence
- Progressive difficulty (speed increases with score)
- Game over screen with final score

## 📁 Files

- `index.html` - Main game page
- `style.css` - Game styling
- `game.js` - Game logic
- `README.md` - This file

## 🚀 Getting Started

Simply open `index.html` in any modern web browser to start playing!

## 💻 Technical Details

- **Canvas Size**: 400x400 pixels
- **Grid Size**: 20x20 pixels
- **Initial Speed**: 150ms per frame
- **Minimum Speed**: 50ms per frame
